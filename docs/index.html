Аудит, що зберігає
природу продукту

Не переробляємо продукт під чужий ідеал. Називаємо, який він є, і знаходимо, де витікає ресурс — щоб закрити діри, лишивши його собою.

Нічого не зберігається. Закрив вкладку — сесія зникла.

Ключ

Підключи ключ Клода

Аналіз робить Клод твоїм ключем. Ключ лишається тільки в цій вкладці (sessionStorage) і зникає, щойно ти її закриєш. Ми його нікуди не передаємо й не зберігаємо.

Немає ключа? Зареєструйся на console.anthropic.com → Settings → API Keys → Create Key. Скопіюй ключ (sk-ant-…) і встав сюди.

Крок 2 / 2 — Аудит

Де витікає ресурс

Проти підтвердженої природи. Кожна теча — як факт, із ціною. Закриття зберігає рідний регістр, не нормалізує продукт.

Дзеркало, не Ворота — називаємо, рішення за тобою. Без аналітики й трекерів.
